# KeyNCheck

Este es nuestro proyecto integrador, el cual se ha trabajado desde hace unos semestre, consta en la optimizacion del tiempo al pase
de lista que realizan los docentes del instituto como tambien salvaguardar los inmuebles de las aulas mediante un sistema de cerradura
electronica y la utilizacion se llevara a cabo mediante la implementacion de lectores para leer el codigo QR de  tarjetas para abrir la 
puerta y registrar la asistencia del dia.
